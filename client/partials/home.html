What is MakeHub?

We’re improving hands-on S.T.E.A.M education by reducing the friction associated with planning educational maker projects. We're doing this by developing on online platform for makers and educators.

HOW TO USE?

In order to use MakeHub, you need to register an account with github.

Register there and then come back to MakeHub to login. This is because we rely on github's infrastructure to build MakeHub so that we don't have to reinvent the wheel. But don't worry! You won't need to open any terminal or write any lines of code.

GitHub is a place that helps people share coding projects. One of its features is forking. It means that anyone can copy your project and start making changes to it to create their own version of it. Github also has a nice logo:

BACKGROUND

Hackidemia is a mobile invention lab that works with kids all over the world. For example, we taught children in Nigeria how to build a solar panel because they didn’t have access to electricity. Or children in Brazil how to build a water filter because they need access to clean water. Since September 2012 we’ve worked with over 3000 children and trained over 200 mentors on 4 continents.

From Hackidemia Brazil

The Makehub online platform provides a layer of meaning and action for the vast world of maker projects found across the internet. The app does this in the following ways:

Users can save projects to their personal lab to share, revisit, and remix later.

Users can create custom kits and buy the required materials for any project directly through Makehub platform.

Our app connects makers and educators with related content, activities, and curriculum from across the web. This is possible through partnerships with open education platforms like Khan Academy and Gooru.

The platform has two components: a browser extension and a web application.
